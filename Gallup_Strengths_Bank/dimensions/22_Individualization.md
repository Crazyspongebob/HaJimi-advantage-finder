# Gallup Theme: Individualization (个别)

## Core Definition
Intrigued with the unique qualities of each person. You have a gift for figuring out how people who are different can work together productively.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 关于“灵魂礼物”的执念。送室友生日礼物，你是否会秘密观察他好几个月，直到你精准地找到那本他一直想要但没舍得买的画册，甚至连包装纸的配色都完美契合他在社交媒体上透露的审美偏好？ `[Signal: Personalized insight]`
- **Topic 2:** 拒绝“被贴标签”。在自我介绍时，对于那些‘你是XX座的吧’或者‘你们XX省的人是不是都爱吃XX’的刻板印象，你是否会本能地感到不快，并迫切地想展示你作为一个独立个体的独特性？ `[Signal: Anti-generalization]`
- **Topic 3:** 朋友圈里的“观察员”。刷朋友圈时，你是否能透过那些精修图，敏锐地察觉到某个朋友今天的情绪其实并不像他表现得那么开心？你是否天生就能读懂每个人的‘隐藏简历’？ `[Signal: Micro-observation]`
- **Topic 4:** 宿舍里的“润滑剂”。如果宿舍里有两个性格完全不对付的小伙伴，你是否会不由自主地观察出他们的性格死角，并像拼图一样把他们放在合适的位置，让他们竟然能和平共处？ `[Signal: Relational orchestration]`
- **Topic 5:** 寻找“闪光点”的面试。社团面试新成员，你是看重那些四平八稳的简历，还是会被那个某一方面强得惊人、哪怕性格有点古怪的‘怪胎’瞬间吸引？ `[Signal: Talent spotting]`
- **Topic 6:** “私人订制”的情感观。你是否认为：一段好的关系，核心标志不在于这种关系有多模范，而在于你们是否为彼此创造了一套全世界唯一的互动方案？ `[Signal: Relational customization]`

### PG Scenario (研究生场景)
- **Topic 1:** 团队组建的“炼金术”。作为课题小组长，你是不是那个能一眼看穿组员A擅长推导、组员B擅长绘图、组员C适合汇报，并能让他们互补得严丝合缝的人？ `[Signal: Complementary teaming]`
- **Topic 2:** 面对导师的“差异化”沟通。你会为了让严厉的导师批准你的课题，而刻意针对他的沟通风格（比如是通过严谨数据还是通过宏阔愿景）去精准设计你的汇报脚本吗？ `[Signal: Style adaptation]`
- **Topic 3:** 学术交流中的“知音”嗅觉。参加学术年会，哪怕在几百个PPT中，你是否也能迅速锁定那个跟你研究品味极度契合的独行侠？你是否觉得自己有一种寻找‘学术灵魂伴侣’的直觉？ `[Signal: Intellectual resonance]`
- **Topic 4:** 对“一刀切”政策的生理厌恶。看到实验室出台那种不管具体情况、强行要大家打卡或者是统一进度的规定，你会因为这些规定抹杀了每个课题组的独特性而感到极度反感吗？ `[Signal: Resistance to standardization]`
- **Topic 5:** 观察论文的“个性墨迹”。看文献时，你是否不仅关注结论，还会琢磨作者在某个论述中的‘个人气质’？你是否喜欢那些带有鲜明个人笔触的硬核研究？ `[Signal: Identifying unique traits]`
- **Topic 6:** “每个人都是一个世界”的准则。是否梦想过未来带出一个能最大程度保留每个成员怪异天赋、并让这些天赋在同一个生态位爆发的‘怪才实验室’？ `[Signal: Visionary pluralism]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我天生就对每个人的独特性、甚至是一些古怪的小癖好感到好奇。 [ ]
2. 我非常擅长根据不同人的性格特点，采用不同的沟通和相处方式。 [ ]
3. 我认为将性格完全不同的人组合在一起工作，能产生最大的创造力。 [ ]
4. 看到一个原本被埋没的独特天赋在我的观察和引导下发光，我会感到很有成就感。 [ ]
5. 很多人认为我是一个非常细腻、周到、且能一眼看穿人心的人。 [ ]
6. 与其使用一套放之四海而皆准的规则，我更喜欢针对每个人的需求制定私人的“例外”。 [ ]
7. 我非常关注各种细微的人际信号和潜在的性格冲突。 [ ]
8. 对我来说，一个毫无个性的“模板化”的人是我觉得最无趣的社交对象。 [ ]
9. 我能迅速地辨别出一个团队中那个最不可替代、最独特的个人价值点。 [ ]
10. “众生皆平等，但个个都不同”是我核心的人性观。 [ ]
11. (反向) 只要能干活就行，没必要去过分了解每个人的个人背景或性格。 [ ]
12. (反向) 碰到复杂的人际关系，我更倾向于公事公办，用死板的流程解决问题。 [ ]
13. (反向) 我觉得所谓寻找每个人的“独特性”其实是在浪费职场社交的成本。 [ ]
14. (反向) 我更喜欢待在那种大家都很像、整齐划一、极其规范化的团队里。 [ ]
15. (反向) 一个人待着比琢磨这群性格迥异的人要轻松得多。 [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极高的人际敏锐度、优秀的团队教练、能让每个人感到被“看见”。适合：个性化咨询、人力资源（高端招聘）、创意总监、复杂的跨国项目协调、家庭教育指导。
- **负面风险**：可能会因为太关注个体的需求而牺牲了团队的通用流程效率、可能会在需要大规模工业化复制的场景下感到痛苦、可能会让那些喜欢标准化管理的下属感到困惑。
- **岗位匹配暗示**：适合需要“深度定制”和“人性洞察”的高价值位置。
