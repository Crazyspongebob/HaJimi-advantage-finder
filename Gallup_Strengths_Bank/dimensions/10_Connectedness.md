# Gallup Theme: Connectedness (关联)

## Core Definition
Faith in the links between all things. There is a reason for everything.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 参与公益时的“蝴蝶效应”感知。你是否深信我们的微小行动（哪怕只是帮舍友带个饭）都在潜移默化地改变这个世界？聊聊你感受到的那种跨越空间的神奇联结。 `[Signal: Cosmic interconnectedness]`
- **Topic 2:** 关于“巧合”的深层解读。在完全陌生的城市遇到老乡，或者是新朋友刚好认识你的小学同学。你是觉得这就是单纯的概率，还是因为缘分带来的直觉而感慨“冥灵之中自有安排”？ `[Signal: Synchronicity awareness]`
- **Topic 3:** 对“磨难”的后期视角。你是否相信即便是一个此时此刻很倒霉、让你想不通的事，也会在未来的某个节点变成一件好事的引证？分享一个你事后才看懂的“反转剧情”。 `[Signal: Redemptive perspective]`
- **Topic 4:** 对远方苦难的真实痛感。面对千里之外的战争、灾害或环境危机，你是否能真实地感到一种身临其境的感觉，觉得“那也是我生命整体的一部分”？ `[Signal: Global empathy]`
- **Topic 5:** 关于宇宙与能量。你会对“万物一体、众生平等”这类讨论产生由衷的共鸣吗？你是否觉得人与自然之间存在某种隐形的、神圣的能量纽带？ `[Signal: Spiritual/Philosophical lean]`
- **Topic 6:** 站在人海中的神奇联觉。当你站在校园的人潮中，是否会产生一种“所有人其实都是连在一起的网”的神奇感觉？这种大局观对你来说是一种慰藉吗？ `[Signal: Collective consciousness]`

### PG Scenario (研究生场景)
- **Topic 1:** 科研中的“命中注定”。遇到死磕不下的难题时，你会觉得这种磨炼是某种必经之路吗？你是否曾从看似无关的跨学科、甚至哲学讨论中，灵光一闪找到了解决实验数据的规律？ `[Signal: Non-linear problem solving]`
- **Topic 2:** 面对科学的敬畏心。在由于对真理的热爱而进行前沿研究时，你是否因为觉得“我正在揭开宇宙运行的某种整体逻辑”，或者“人类所有知识都是连在一起的”而感到一种由于崇高感而产生的震撼？ -> NO! FIX!
- **Topic 2:** 面对科学的敬畏心。在进行前沿研究时，你是否因为觉得“我正在揭开宇宙运行的某种整体逻辑”，或者“人类所有知识都是连在一起的”而感到一种崇高感带来的大震撼？ `[Signal: Transcendent motivation]`
- **Topic 3:** 跨学科融合的直觉。当不同背景的团队聚在一起时，你是否是那个最快发现“虽然术语不同，但底层逻辑其实是互通的”的人？这种打破壁垒的愿望强烈吗？ `[Signal: Integrative vision]`
- **Topic 4:** 超越派系纷争的视角。面对学术圈的种种隔阂，你是否能从更高维度看到这些内耗的短视？你是否深信大家本质上都在为同一座智慧大厦添砖加瓦？ `[Signal: Conflict de-escalation]`
- **Topic 5:** 与历史巨人的跨时空对话。你会因为“某篇几十年前的古老文献突然完美解答了你当下的困惑”，而因为这种跨越时空的联结感而感到内心澎湃吗？ `[Signal: Temporal connectedness]`
- **Topic 6:** 科研的终极归宿。你是否觉得你的成果不仅属于你的，也是属于全人类甚至全宇宙运行逻辑的一部分，是这种“大我”感在支撑你工作吗？ `[Signal: Purpose beyond self]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我相信世界上没有纯粹的偶然，万事万物皆有其因缘联系。 [ ]
2. 我常常能在看似毫不相关的现象之间发现深刻的、内在的一致性。 [ ]
3. 我认为人类是一个整体，每个人的行为都会对全局产生影响。 [ ]
4. 这种“大局观”或“全局感”能让我在面临困难时感到内心的宁静。 [ ]
5. 我倾向于从更宏大、更长远的角度来看待眼前的琐碎问题。 [ ]
6. 我相信任何生命形式之间都存在着深层的灵性联结。 [ ]
7. 这种“万物一体”的信念驱使我去做更多利他的事情。 [ ]
8. 当我感到孤立无援时，我会提醒自己：我始终是这个庞大宇宙的一部分。 [ ]
9. 我非常尊重那些探索生命本质、宇宙奥秘的理论或哲学。 [ ]
10. “利他就是利己”是我深以为然的处世哲学。 [ ]
11. (反向) 宇宙是冷漠且随机的，没有任何预设的意义. [ ]
12. (反向) 我认为每个人都是独立的个体，别人的痛苦和我毫无关系. [ ]
13. (反向) 所谓的“因果”只是人类强行加在混乱世界上的幻觉. [ ]
14. (反向) 我只关注眼前的、可见的利益，不相信虚无缥缈的联系. [ ]
15. (反向) 我认为世界是由孤立的碎片组成的，没有任何整体性可言. [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极高道德高度、跨学科思维、宽容度极高、能给团队带来终极意义感。适合：环境科学、生命哲学研究、大型社会公益管理、多元文化协调工作。
- **负面风险**：在极度现实且零和博弈的环境中可能会显得“不切实际”、可能会因为追求整体性而忽略了微观的具体矛盾。
- **岗位匹配暗示**：适合需要“大局观” and “价值感”的领域。
