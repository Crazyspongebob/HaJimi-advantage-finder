# Gallup Theme: Analytical (分析)

## Core Definition
“Prove it. Show me why what you are claiming is true.”

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 面对老板或导师那种模糊的夸奖（比如“做得不错”）。你是那种会开心起飞的人，还是会忍不住下意识追问“具体好在哪？数据提升了多少？”。如果你发现对方给不出具体论据，你会觉得这种夸奖并没有什么价值吗？ `[Signal: Critique of subjective praise]`
- **Topic 2:** 看到那种“震惊！XX专业未来工资最高”的热门帖子。你的第一反应是转发吃瓜，还是会先去找它的调研样本量、数据来源以及发布机构的背景？聊聊你的“信息过滤”习惯。 `[Signal: Source skepticism]`
- **Topic 3:** 选课或买大件数码产品时的决策过程。你是否非要拉出一张“参数对比表”，看清楚每一项底层逻辑差异才敢下单？分享一个你通过对比找到“最优解”的经历。 `[Signal: Comparative logic]`
- **Topic 4:** 朋友失恋哭诉“他再也不爱我了”时的应对模式。你是会拍拍对方给予情绪安慰，还是会忍不住开启“侦探模式”帮她复盘：‘他周一的冷漠和周二的迟到之间是不是有某种必然联系’？聊聊你的“情绪拆解”习惯。 `[Signal: Emotional deconstruction]`
- **Topic 5:** 课堂上发现老师讲错了一个逻辑点。你是那种“算算了”的佛系学生，还是会非要下课拿着公式去跟老师辩论清楚、追求逻辑绝对精准的人？这种对“对错”的执念对你来说重吗？ `[Signal: Precision drive]`
- **Topic 6:** 关于娱乐方式的审美。比起纯靠运气和直觉的桌游（如大富翁），你是否更喜欢那种需要严密逻辑推演、博弈和赛后复盘的互动（如剧本杀复盘、狼人杀）？ `[Signal: Entertainment preference]`

### PG Scenario (研究生场景)
- **Topic 1:** 读到一篇顶刊文章，哪怕学术界都在赞美。你会不会还是忍不住怀疑它的实验设计有没有漏洞？或者一定要亲手复现一下它的模型参数和原始数据才敢真正相信？ `[Signal: Peer skepticism]`
- **Topic 2:** 当实验结果预期严重不符时。你第一反应是感到困惑，还是兴奋地觉得“这背后绝对有某种隐藏的逻辑变量我没发现，我要把它强行拆开看个究竟”？聊聊你这种“死磕真相”的状态。 `[Signal: Pursuit of hidden structure]`
- **Topic 3:** 在组会上讨论方案。如果你发现导师给出的一个结论在逻辑上存在哪怕一丁点的不严密，你会为了逻辑自洽而顾不上情面地“穷追猛打”吗？ `[Signal: Intellectual integrity]`
- **Topic 4:** 面对杂乱无章、多达几万行的原始数据。你是会感到由于头重脚轻而产生的恐惧感，还是会有一种“终于可以抽丝剥茧、找出底层规律”的解谜快感？ `[Signal: Data affinity]`
- **Topic 5:** 对于那种宣称“靠直觉告诉我这里该这么调参数”的技术大牛。你会心生崇拜，还是下意识觉得“这种不可量化的经验极其不科学、不严谨”？ `[Signal: Methodology bias]`
- **Topic 6:** 关于世界的认知。你是否认为，任何复杂的人类情感或社会现象，背后都有一个可以用数学或逻辑解释清楚、可以被建模的底层模型？ `[Signal: Rational reductionism]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 在没有看到确凿的数据或事实之前，我很难被说服去相信一个结论。 [ ]
2. 我喜欢将复杂的问题拆解成小的、可管理的逻辑组成部分。 [ ]
3. 发现事物背后的潜在规律和因果关系，对我来说很有吸引力。 [ ]
4. 我更倾向于客观、中立地看问题，尽量避免情绪化的干扰。 [ ]
5. 我认为逻辑和理性和解决任何挑战的最核心工具。 [ ]
6. 所有的结论都必须经得起严密的逻辑推导和验证。 [ ]
7. 我对各种事物的矛盾描述、缺乏证据支撑的人感到很沮丧。 [ ]
8. 在决策时，我宁愿相信冰冷的数据，也不愿相信火热的直觉。 [ ]
9. 我认为理智是人类最强大的武器，我愿意时刻打磨它。 [ ]
10. 当我发现一个错误的逻辑漏洞时，我有强烈的冲动去指正它。 [ ]
11. (反向) 跟着感觉走通常比苦思冥想更有效. [ ]
12. (反向) 我认为生活中很多事是不需要逻辑的，糊涂一点更好. [ ]
13. (反向) 过度分析会扼杀创造力和生活的乐趣. [ ]
14. (反向) 相比起严密的论证，我更容易被感人的故事所打动. [ ]
15. (反向) 我认为数据只是工具，经验和直觉才是真正的大脑引擎. [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极高客观性、穿透表象的洞察力、严密的防御逻辑。适合：算法研发、IC设计、数据分析、合规审查。
- **负面风险**：可能显得有些“冷冰冰”、在需要直觉决策的时刻可能犹豫不决、会对同伴产生社交压迫感。
- **岗位匹配暗示**：适合需要“绝对准确性” and “深层因果推导”的岗位。
