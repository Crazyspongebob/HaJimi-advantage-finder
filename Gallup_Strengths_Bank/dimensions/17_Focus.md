# Gallup Theme: Focus (专注)

## Core Definition
Taking a direction, following through, and making necessary corrections to stay on track.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 闹市里的“真空罩”。如果自习室有人在聊八卦、刷短视频，你一戴上耳机是否就能瞬间切断与外界的物理联系，让全世界只剩下你眼前的这几个理论模型？ `[Signal: Concentration intensity]`
- **Topic 2:** 拒绝“节外生枝”的利落。当你在赶作业时，如果死党突然喊你去吃火锅或看电影，你是不是那种能瞬间判断这会打破节奏、并能非常自然（甚至冷酷）地秒回“没门儿”的人？ `[Signal: Priority protection]`
- **Topic 3:** 目标的“终局感”。当你定下要考出那门极难的证书，你是否会在脑子里自动过滤掉所有与之无关的信息，每天睁眼第一件事就是确认“我离那个红点还差多远”？ `[Signal: Goal-directed persistence]`
- **Topic 4:** 讨厌“多线操作”。比起同时开五个PPT窗口，你是不是更享受那种‘今天下午我只死磕这一件事，把它干到底’的线性快感？ `[Signal: Single-tasking preference]`
- **Topic 5:** 偏差修正的“罗盘”。如果在推进大计划时出现了一点小意外，你是会陷入混乱，还是会通过快速微调，迅速确保船头依然对准最初那个唯一的靶心？ `[Signal: Course correction]`
- **Topic 6:** 关于“效率”的偏执。你是否认为，任何没有明确指向（End Goal）的会议或讨论，都是对你生命的一种亵渎？ `[Signal: Efficiency as purpose]`

### PG Scenario (研究生场景)
- **Topic 1:** 核心变量的“守望者”。钻进子课题后，你会不会一整周都只盯着那个核心矛盾，甚至连饭都忘了吃？如果别人建议加个无关痛痒的支线，你第一反应是不是‘滚，别弄乱我的主线’？ `[Signal: Mainstream maintenance]`
- **Topic 2:** 枯燥节点的“长跑定力”。面对那需要连续几个月才有一次反馈的冷门实验，你是否能忍受这种漫长的寂静，始终如一日地死盯着那个微小的变量变化？ `[Signal: Long-term focus endurance]`
- **Topic 3:** 对“智力噪音”的极端排斥。当大家都在讨论跨界合作、跨学科横向项目时，你会不会因为这会分散你的核心学术产出而感到一种深深的警惕？ `[Signal: Noise filtering]`
- **Topic 4:** “闭关”式的爆发。项目到了收官阶段，你能够做到整周不出实验室、不回微信，只为了在那个唯一的切口上爆发出最强的压强，实现学术突破吗？ `[Signal: Focused burst capacity]`
- **Topic 5:** 进度的“标尺”。你大脑里是否始终挂着一张经纬度极其密集的进度表，任何细微的航道偏离都会让你产生强烈的纠偏冲动？ `[Signal: Real-time alignment]`
- **Topic 6:** 你的“狙击手”理想。是否梦想过成为那个在领域内专攻最硬的骨头、一旦锁定目标就绝不脱靶、且每次都能精准击中那个关键“阿基米德点”的人？ `[Signal: Precision execution ambition]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 只要我选定了方向，我就很难被琐事分心。 [ ]
2. 如果一件事没有一个明确且唯一的目标，我会感到无所适从。 [ ]
3. 我认为“一心一意”比“多线操作”更有效率、也更让我愉悦。 [ ]
4. 我具备一种在嘈杂或高压环境下依然能沉浸在工作中的能力。 [ ]
5. 达成那个具体目标的过程，对我来说比任何事情都重要。 [ ]
6. 我非常擅长从繁杂的信息中一眼挑出那条最关键的主线。 [ ]
7. 我认为所谓的“变通”如果没有指向核心目标，其实就是一种逃避。 [ ]
8. 对我来说，截止日期不是压力，而是我保持专注的动力。 [ ]
9. 我能迅速地辨认出一个方案中那个最具杠杆效应的关键目标。 [ ]
10. “不达目的誓不罢休”是我深以为然的成功哲学。 [ ]
11. (反向) 碰到新鲜有趣的点子，我很容易就会被带偏，忘了最初在干嘛。 [ ]
12. (反向) 我认为生活就该随兴而发，制定死板的唯一目标会让我感到窒息。 [ ]
13. (反向) 我更喜欢同时处理多项任务，这样能让我感到充实。 [ ]
14. (反向) 我觉得那些整天盯着一个目标看的人，其实是缺乏格局的表现。 [ ]
15. (反向) 我通常只有在压力极大时才能集中注意力。 [ ]
