# Gallup Theme: Competition (竞争)

## Core Definition
Comparison to others. You want to win and be the best.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 期末出分时的“排名焦虑”。如果你拿个A但发现全班有一半人也拿了A，甚至有人分比你更高，你的满足感会不会瞬间打折？聊聊你对“夺冠”和“第一名”的真实渴望。 `[Signal: Ranking fixation]`
- **Topic 2:** 玩游戏或参加比赛时的心态。你是否觉得“友谊第一”这种话对自己无效，只要上场目标就是拿那个冠军奖杯？分享一次你为了“赢”而拼尽全力的热血经历。 `[Signal: Win-loss parity]`
- **Topic 3:** 相对产出的微妙心理。如果一个项目你做得很好，但发现周围有人拿到的奖金或荣誉更多，你的成就感会因此降低吗？你是否习惯通过他人的位置来定位自己的进步？ `[Signal: Relative performance]`
- **Topic 4:** 操场的“反超本能”。在跑步时，如果旁边有人轻松越过了你，你会不会不自觉地加快速度、非要凭着那股好胜心反超回来才觉得顺气？这种不请自来的竞技感频繁吗？ `[Signal: Spontaneous rivalry]`
- **Topic 5:** 对“大神榜单”的反应。看到同领域的顶尖人物排行榜，你的第一反应是感到“被压制”的痛苦压力，还是因为有了明确的赶超目标而感到兴奋？ `[Signal: Benchmarking response]`
- **Topic 6:** 关于“对手”的必要性。你是否觉得，没有强劲对手的成功其实挺无味的？你更享受那种在严酷的“丛林规则”中杀出重围、最终登顶的快感吗？ `[Signal: Need for opposition]`

### PG Scenario (研究生场景)
- **Topic 1:** 得知同期的那个谁发了顶刊。你是真诚祝福他“牛啊”，还是心里暗暗憋了一口气：‘不，我得赶紧写出那一篇，我绝不能落在他后边’？聊聊这种被同伴激发出的斗志。 `[Signal: Social comparison drive]`
- **Topic 2:** 在实验室里对“学术指标”的敏感度。你是否特别关注引用指标、H因子排名或者组内的产出排名？领先于同龄人带给你的快感有多强烈？ `[Signal: Metric competition]`
- **Topic 3:** 面对极具挑战但名额极其有限的机会（如国家奖学金）。这种“赢家通吃”的规则会不会不仅不让你害怕，反而让你像打了鸡血一样兴奋？ `[Signal: Scarcity motivation]`
- **Topic 4:** 学术会议上的“主张权”。在提问环节，你是否倾向于展示出比对方更深厚的见解，以确立你在该领域的学术地位？聊聊这种对地位的隐形争夺感。 `[Signal: Intellectual dominance]`
- **Topic 5:** 导师的赞赏分配。如果导师在组会上公开夸奖了别的同门，你会不会立刻反思：‘我哪点做得不如他？我下次一定要拿出一份更出彩的结果让他刮目相看’？ `[Signal: Validation rivalry]`
- **Topic 6:** 评价一个科研人员好不好的标准。你是否认为，评价一个人的唯一标准，就是他在顶级赛道（如名刊发表量）上的硬核排名？ `[Signal: Absolute hierarchy]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我通常会通过观察别人的表现来衡量自己的进步。 [ ]
2. 赢得比赛或获得最高名次对我来说非常重要。 [ ]
3. 将挑战视为一场比赛，能极大地激发我的工作动力。 [ ]
4. 如果我知道别人和我做同样的事而且做得更好，我会感到很不自在。 [ ]
5. 我非常享受在公平竞争中战胜对手的感觉。 [ ]
6. 衡量成功的唯一标准就是我是否超过了竞争对手。 [ ]
7. 我天生就有一种想要在各方面都表现得比别人更出色的冲动。 [ ]
8. 对我来说，拿第二名往往意味着失败。 [ ]
9. 这种与他人的对比，能让我保持长期的兴奋和斗志。 [ ]
10. 我非常关注各种公开的排名和评价体系。 [ ]
11. (反向) 我认为最重要的是超越昨天的自己，而不是超越别人. [ ]
12. (反向) 我觉得和别人竞争是一件非常累、且毫无意义的事情. [ ]
13. (反向) 为了团队和谐，我宁愿隐藏自己的锋芒，不让别人感到尴尬. [ ]
14. (反向) 我认为只要自己尽力了，结果是什么样我都能坦然接受. [ ]
15. (反向) 相比起赢过别人，我更关注工作的社会价值和个人成长. [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极高爆发力、强烈的结果导向、永不言败的斗志。适合：销售、竞技体育、高频交易、处于快速扩张期的初创企业。
- **负面风险**：容易造成人际关系紧张、过度关注对手而迷失战略重心、输掉时可能产生较大的心理落差。
- **岗位匹配暗示**：适合“赢者通吃”的赛道及高压力KPI环境。
