# Gallup Theme: Developer (伯乐)

## Core Definition
Recognizing the potential in others and helping them improve.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 校园里的“潜力股”捕手。如果有个新同学做什么都笨手笨脚、甚至被大家私下嫌弃，你是否会本能地想：“他只是还没找到合适的频率，如果我能带他一段时间，他准能惊艳所有人”？ `[Signal: Potential spotting]`
- **Topic 2:** 养成系的“成就感”。当你帮室友从零开始修改简历、带他找准方向，最后看到他拿到Offer后的那一刻，你是否会感到一种比自己拿奖还要强烈的愉悦？你是否迷恋这种见证他人蜕变的过程？ `[Signal: Mentorship fulfillment]`
- **Topic 3:** “微光”放大器。在评价一份很不成熟的小组方案时，你是否能精准地避开所有漏洞，先一把抓住那个极其有灵气的小细节大加赞赏，试图用这点火花点燃对方的斗志？ `[Signal: Micro-progress recognition]`
- **Topic 4:** 见证逆袭的“云陪伴”。你会因为看到一个性格孤僻的朋友终于敢在公开场合大声说话，而感到一种由衷的欣慰吗？你是否会因此而对他刮目相看，并期待这种进步而带来的更多可能性？ `[Signal: Empowerment resonance]`
- **Topic 5:** 寻找人性的“成长杠杆”。你是否会情不自禁地在心里给身边的每个人排演“进化路径”？比如看到那个很幽默但有点邋遢的哥们，会想：“他要是去搞自媒体绝对能成大器”？ `[Signal: Potential mapping]`
- **Topic 6:** “养成系”团队的执念。如果可以选择，你是更愿意加入一个现成的‘全明星’豪华战队，还是更享受那种带着一群什么都不会的后辈、看着他们在你手里飞速成长并最终横扫全场的快感？ `[Signal: Nurturing preference]`

### PG Scenario (研究生场景)
- **Topic 1:** 实验室里的苏格拉底。师弟问你某个算法怎么写，你明明一分钟能教完，却非要花半小时问他三个问题引导他自己想明白，你是否特别享受那个“他眉头舒展、眼神明亮”的瞬间？ `[Signal: Socratic guidance]`
- **Topic 2:** “炸机”后的心理托底。看到新来的同门弄坏了昂贵的实验仪器，大家都在抱怨时，你会不会主动走过去说‘别怕，我当年弄坏得比你还狠’，并趁机帮他分析底层原理？ `[Signal: Resilience fostering]`
- **Topic 3:** 看着雏鹰起飞的豁达。如果你的研究助理通过你的指导去往了比你更高的平台，你会感到被超越的焦虑，还是会有一种‘不愧是我一手带出来的’那种由衷的满足？ `[Signal: Success through others]`
- **Topic 4:** 对“大器晚成”的长期持仓。你会因为发现某个师妹现在的产出虽然少，但逻辑根基极其扎实，而坚定地看好她未来的爆发，并愿意给予她比别人更多的耐心和关注吗？ `[Signal: Patience with growth]`
- **Topic 5:** 寻找研究热忱的“调音师”。你是否发现，只要你稍微调整一下对不同成员的反馈方式（比如引导、反问或并肩作战），就能神奇地激活他们体内沉睡的研究潜能？ `[Signal: Motivational customization]`
- **Topic 6:** 你的“名师”理想。是否梦想过当你退休时，学术界顶尖的一批人才都曾在那段最懵懂、最艰难的时光里，感激过你那如同春风化雨般的引导之恩？ `[Signal: Legacy via people]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我不仅关注他人的现状，更关注他们未来发展的潜力。 [ ]
2. 看到别人在自己的帮助下取得进步，我会感到巨大的成就感。 [ ]
3. 我非常乐于投入时间和精力去辅导和培养他人。 [ ]
4. 我擅长发现每个人身上那一丁点儿进步的迹象。 [ ]
5. 与表现卓越的人共事固然好，但我更喜欢带一个有潜力的新人。 [ ]
6. 我总是能给那些失去信心的人带去希望和具体的改进建议。 [ ]
7. 我认为评价一个领导者好不好的标准就是他培养出了多少人才。 [ ]
8. 我能看到别人自己都没有察觉到的闪光点。 [ ]
9. 我非常享受那种“陪伴并见证成长”的过程。 [ ]
10. “诲人不倦”的耐心是我性格中最引以为傲的一部分。 [ ]
11. (反向) 那些表现平庸的人不值得我浪费宝贵的时间去培养。 [ ]
12. (反向) 我认为职场就是优胜劣汰，不行就该被淘汰，没必要辅导。 [ ]
13. (反向) 我更在乎我个人的技术产出，别人的进步与我无关。 [ ]
14. (反向) 如果一个人多次教不会，我就会彻底对他失去信心。 [ ]
15. (反向) 我认为天才是不需要培养的，庸才是培养不出来的。 [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极佳的教练潜质、能极大提升团队凝聚力、善于挖掘非对称人才。适合：HR培训教官、教育行业、跨代际团队管理者、技术导师。
- **负面风险**：可能会在不值得培养的人身上浪费过多精力、可能会过度关注他人成长而忽略了自己的KPI考核。
- **岗位匹配暗示**：适合需要“人带人”和“组织发展”的赛道。
