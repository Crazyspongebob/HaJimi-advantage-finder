# Gallup Theme: Belief (信仰)

## Core Definition
Having core values that are unchanging. These values give life purpose and direction.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 如果小组作业里有个拿高分的“捷径”（比如修饰一点点问卷数据）。哪怕最后会失去拿优的机会，你会是为了坚守“学术诚实”而宁愿拿B的人吗？聊聊你心中那种“不为利益折腰”的瞬间。 `[Signal: Ethical core]`
- **Topic 2:** 选择社团或社交圈的标准。是更看重对方能不能带你飞（利益），还是更看重价值观的共振？如果理念不合，你会觉得再牛的人也就是“道不同不相为谋”吗？ `[Signal: Value-based social filtering]`
- **Topic 3:** 你心中那些“不能触碰的底线”。哪怕周围所有人都在为了方便而逾越它，你的内心是否也有一种强烈的抗拒感在拉扯？那是什么原则在守护你？ `[Signal: Unyielding boundaries]`
- **Topic 4:** 如果发现参加的志愿活动或项目存在猫腻。你会为了证书睁一只眼闭一只眼，还是会立刻因为那种“生理性恶心”而选择退出？聊聊你的道德洁癖。 `[Signal: Moral integrity]`
- **Topic 5:** 关于“人为什么活着”的看法。你是否深信人活着必须是为了点什么（比如爱、真理、责任），否则赚再多钱也就是“行尸走肉”？那种让你感到“生活有火苗”的意义感在哪？ `[Signal: Existential meaning]`
- **Topic 6:** 逆流而上的勇气。当家人反对你选那个“你认为很有意义”但“没钱途”的专业时，你是屈服于现实，还是会为了那份使命感而爆发惊人的反抗精神？ `[Signal: Mission-driven resistance]`

### PG Scenario (研究生场景)
- **Topic 1:** 面对月薪翻倍但没啥社会贡献的Offer，和月薪微薄但实实在在科研利民的项目。你会因为那种“我想做点有意义的事”的直觉而选后者吗？聊聊你的选择取舍。 `[Signal: Sacrifice for meaning]`
- **Topic 2:** 当研究结论可能会被商业力量误用或歪曲时。你会为了得罪资助人的代价，也要在论文里大声疾呼出真实的局限和风险吗？聊聊你对“真实”的坚持。 `[Signal: Academic truth-seeking]`
- **Topic 3:** 支撑你在深夜实验室熬下去的终极力量。你觉得让你坚持下来的是未来的高薪，还是某种“我必须为这个行业/社会做点什么”的使命感？ `[Signal: Internal fire]`
- **Topic 4:** 面对学术圈的“灰色地带”（如过度挂名、资源置换）。你会感到极度不适并选择做一个“打破沉默”的人吗？你是否看重这种纯粹的程序正义？ `[Signal: Justice drive]`
- **Topic 5:** 哪怕领域很冷门、没人看好。你内心的坚定是否从未动摇，因为你深信它的长远价值？那种“板凳甘坐十年冷”的底气从何而来？ `[Signal: Visionary perseverance]`
- **Topic 6:** 关于重大决策的初心。你是否习惯在所有复杂衡量之后问自己一句：“这件事符合我当初踏入这里的初衷吗？”聊聊你的价值观锚点。 `[Signal: Value-first decision making]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我有一套坚定的核心价值观，它们是我所有行动的指南。 [ ]
2. 即使面临巨大的压力或诱惑，我也不愿意牺牲自己的原则。 [ ]
3. 我追求那些对我来说非常有意义的工作，而不仅仅是金钱或名声。 [ ]
4. 当我的价值观受到挑战时，我会表现出极强的反抗精神。 [ ]
5. 如果一份工作与我的个人信仰产生冲突，我很难在那待下去。 [ ]
6. 我相信人生的价值在于为他人或社会做出客观贡献。 [ ]
7. 我对那些为了利益而放弃底线的人感到深恶痛绝。 [ ]
8. 这种“内在的确定性”是我遇到困难时最大的精神支柱。 [ ]
9. 我经常会为了维护我认为正确的事情而选择站在多数人的对面。 [ ]
10. “不忘初心”对我来说不仅仅是口号，而是贯穿始终的行为准则。 [ ]
11. (反向) 只要利益足够大，我可以灵活地调整自己的立场和底线。 [ ]
12. (反向) 我认为所谓的“信仰”或“原则”在现实竞争面前往往是苍白的。 [ ]
13. (反向) 我是一个极度务实的人，哪种选择对我最划算我就选哪种。 [ ]
14. (反向) 如果大家都在逾矩，我没有必要非要做那个圣人。 [ ]
15. (反向) 工作的意义就是赚钱养家，其他的社会价值都是虚的。 [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极高忠诚度、极强稳定性、使命感驱动。适合：公益机构、基础科学研究、国家战略保障部门、合规风控。
- **负面风险**：可能显得有些“死板”、在需要变通的商业环境下可能感到痛苦、容易和上司发生价值观冲突。
- **岗位匹配暗示**：适合需要“信仰”和“长期坚持”的重器岗位，避免纯粹的投机性岗位。
