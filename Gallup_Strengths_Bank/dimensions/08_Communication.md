# Gallup Theme: Communication (沟通)

## Core Definition
Putting thoughts into words. You are good conversationalists and presenters.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 大家都害怕的上讲台做Pre时刻。你是否反而会把它看作一次“个人秀”的最佳舞台？比起干巴巴念完PPT，你是不是更在意能不能讲一个让全场爆笑的段子或形象的比喻？ `[Signal: Performance joy]`
- **Topic 2:** 关于你的“表达原始驱动力”。如果你没戴耳机听歌，是否非要跟旁边人说点什么才觉得自在？你能把一件普通的比如“昨晚吃的兰州拉面”讲得波澜壮阔、引人入胜吗？ `[Signal: Verbal drive]`
- **Topic 3:** 你的表达风格偏好。是更习惯用一长串文字严谨解释，还是更习惯发长语音、打视频，甚至配上夸张的肢体动作来表情达意？聊聊你最舒服的传达方式。 `[Signal: Expressive channel preference]`
- **Topic 4:** 群聊里的“灵魂人物”。当大家都在沉默收信息或“收到”复读时，你是否是那个最爱活跃气氛、能用精准表情包和风趣回复带起讨论节奏的人？ `[Signal: Content generation]`
- **Topic 5:** 职业病式的表达观察。刷短视频或看演讲时，你是不是会下意识拆解对方的台词对比、节奏控制，甚至会忍不住在镜子前模仿一段？ `[Signal: Communication sensitivity]`
- **Topic 6:** 关于“语言的魔力”。你是否深信任何复杂的事情只要“聊透了、说开了”就能解决？你对那种通过对话达成共鸣的过程有多大的迷恋？ `[Signal: Faith in dialogue]`

### PG Scenario (研究生场景)
- **Topic 1:** 向大老板（导师/院长）汇报枯燥的进度。你是习惯列出一堆公式和技术指标，还是更擅长用很形象的比喻，把晦涩的算法讲得通俗易懂、连文科生都能听明白？ `[Signal: Analogical thinking]`
- **Topic 2:** 跨学科的分支讨论。你是否是那个能把晦涩的实验室专业词汇翻译成“人话”、让不同背景的人也能瞬间get到你研究价值的“学术翻译官”？ `[Signal: Knowledge translation]`
- **Topic 3:** 互动的快乐。比起一个人默默在书斋写论文，你是否觉得“把它讲给别人听”、甚至是“在汇报中被提问互动”的过程更能激发你的思考灵感？ `[Signal: Interactive reflection]`
- **Topic 4:** 引导社交僵局。面对实验室里那些性格孤僻的数据天才，你是否能通过你的谈吐引导他们开口，并把他们的价值优雅地呈现给全组？ `[Signal: Facilitation skill]`
- **Topic 5:** 答辩现场的“太极功”技巧。面对评委的犀利刁难，你能否通过巧妙的辞令优雅地化解尴尬、甚至把对方带进你的逻辑舒适区？聊聊这种临场表达的敏捷度。 `[Signal: Rhetorical agility]`
- **Topic 6:** 关于未来的科普理想。你是否梦想过把冰冷的实验室数据变成有温度的故事（比如做个硬核科普大V）？聊聊你对这种“知识传播”的野心。 `[Signal: Narrative ambition]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我发现自己很容易就能把脑子里的想法转化为清晰、生动的语言。 [ ]
2. 我非常享受在公共场合进行演讲或分享。 [ ]
3. 我认为讲故事和使用比喻是传达复杂信息最有效的方式。 [ ]
4. 我发现语言文字本身就带带有某种让我兴奋的力量。 [ ]
5. 很多人认为我是一个非常健谈且极具感染力的人。 [ ]
6. 与其默默思考，我更喜欢一边说一边理清思路。 [ ]
7. 我非常关注字词的使用、节奏感以及表达的情绪价值。 [ ]
8. 我能迅速地根据听众的反馈调整自己的说话内容和方式。 [ ]
9. 我认为“被听见”和“被理解”是工作中最大的奖赏。 [ ]
10. 对我来说，一个无法被精彩表达出来的 Idea 是不完整的。 [ ]
11. (反向) 我觉得和人深度交流是一件非常消耗能量的事情。 [ ]
12. (反向) 对于复杂的问题，我宁愿写报告也不想当众解释。 [ ]
13. (反向) 我通常会因为口误或冷场而感到极度沮丧和尴尬。 [ ]
14. (反向) 相比起口头表达，我更信任严谨的实验结果而非修饰过的辞令。 [ ]
15. (反向) 我在人群中通常更倾向于做一个倾听者，而不是发言者。 [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极强感染、信息传递效率高、优秀的公共关系能力。适合：PR公关、自媒体博主、培训师、科学普及岗、跨部门协调员。
- **负面风险**：可能显得有些“华而不实”、如果缺乏内容支撑会显得浅薄、容易在沉默的专注工作中感到枯燥。
- **岗位匹配暗示**：适合需要“外向输出”和“内容二创”的赛道。
