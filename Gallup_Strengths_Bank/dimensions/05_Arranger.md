# Gallup Theme: Arranger (统筹)

## Core Definition
Organizing all the pieces and resources to get maximum productivity.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 团建定车、定餐、定行程这种“乱成一锅粥”的场面，你是觉得头大还是觉得这种“拼图游戏”很有趣？聊聊你如何盘活各种资源，并在这个过程中根据大家状态随时灵活调整。 `[Signal: Resource orchestration]`
- **Topic 2:** 宿舍联手搞大项目时的分工习惯。你是那个能把每个人该干嘛安排得明明白白，且还能随时“见招拆招”调整任务的人吗？分享一个你通过动态分工盘活全组的故事。 `[Signal: Dynamic team allocation]`
- **Topic 3:** 关于效率的执念。旅行时你是追求“时间利用率最大化”的严丝合缝派，还是那种觉得“只要逻辑顺了怎么都行”的人？你会为了省下半小时而重新排列行程吗？ `[Signal: Efficiency optimization]`
- **Topic 4:** 跨院系合作遇到障碍时的脑筋急转弯。如果对方不配合，你是死磕现有方案，还是会立刻跳出来换一种组合方式（比如找个更有空的部门）来绕道实现目标？ `[Signal: Configuration flexibility]`
- **Topic 5:** 关于“变数”的兴奋感。当新的变量（比如突发通知）打乱原有节奏时，你是感到挫败，还是会有一种“最好的安排永远在下一秒”的重组兴奋感？ `[Signal: Response to new variables]`
- **Topic 6:** 你的电脑或手机分类系统。是不是有一套连你自己都佩服的复杂逻辑，能让你在瞬间调动所需的一切？聊聊你习惯怎么通过“分类”来理顺世界。 `[Signal: Systematization]`

### PG Scenario (研究生场景)
- **Topic 1:** 实验室仪器冲突、合作单位对接样品……在这种多线并行的手忙脚乱中，你是否反而能理出一条最省时间的“最优路径”让大家都没意见？聊聊你的多线程平衡术。 `[Signal: Multi-threaded coordination]`
- **Topic 2:** 导师扔来一堆乱七杂八的子项目（写材料、做仿真、带本科生）。你会觉得崩溃，还是觉得“这么多资源在我手里，我可以组一盘大局”？谈谈你对复杂局面的管理欲望。 `[Signal: Complexity management]`
- **Topic 3:** 跨学科项目的流程优化。你是否总能一眼看出：这个模块应该先做，那个可以同步，从而大幅缩短研发周期？这种对“流程”的直觉对你来说普遍吗？ `[Signal: Process streamlining]`
- **Topic 4:** 经费有限时的“省钱艺术”。你会不会通过“共享、置换、或寻找平替”的方法，把原本不够用的钱花出双倍效果？分享一个你通过整合资源化腐朽为神奇的经历。 `[Signal: Resource resourcefulness]`
- **Topic 5:** 项目组增加新成员时的第一反应。你会第一时间思考“他放哪个位置最能提升全组效率”吗？聊聊你脑子里那种“人力资源动态图”。 `[Signal: Human resource configuration]`
- **Topic 6:** 你的高效率工作状态。是否习惯一边跑实验、一边写文档，同时耳机里还听着讲座？你觉得这种“多线程并行”才是最自然的状态吗？ `[Signal: Multitasking comfort]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我擅长处理许多同时发生的任务或变量。 [ ]
2. 我能够快速识别并整合各种资源（人、财、物）来实现目标。 [ ]
3. 我喜欢根据新的信息即时调整计划，哪怕这意味着之前的安排要打乱重来。 [ ]
4. 当情况变得复杂多变时，我通常能理出最高效的行动轨迹。 [ ]
5. 看到原本杂乱无章的碎片被我组织成一个高效运转的整体，我会感到很有成就感。 [ ]
6. 我认为“灵活配置”比“死守计划”更能带来生产力。 [ ]
7. 我能一眼看出在一堆乱七杂八的事情中，如何安排先后顺序最科学。 [ ]
8. 在团队中，我通常是那个决定“谁干什么”的最合适人选。 [ ]
9. 我非常享受那种多目标、多线程协同推进的挑战感。 [ ]
10. 我能够根据项目的实时进展，迅速切换人员分工和资源投入。 [ ]
11. (反向) 面对多个突发任务并存时，我会有强烈的混乱感和无助感。 [ ]
12. (反向) 我倾向于一次只专注做一件事，做完再做下一件。 [ ]
13. (反向) 频繁的计划变动会让我感到非常愤怒和不安全。 [ ]
14. (反向) 我认为团队的分工一旦确定就不应更改，否则会乱套。 [ ]
15. (反向) 我更喜欢别人把所有东西都准备好、我只负责执行其中一环。 [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极强综合能力、优秀的指挥官潜质、能将有限资源价值最大化。适合：项目经理、复杂供应链管理、系统架构师。
- **负面风险**：可能让队友觉得计划“变来变去”、可能因为追求极致效率而忽略了人文关怀。
- **岗位匹配暗示**：适合需要“盘活整局”的中枢岗位。
