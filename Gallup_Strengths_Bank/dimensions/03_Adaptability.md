# Gallup Theme: Adaptability (适应)

## Core Definition
Living in the moment. Seeing the future as a destination created by today's choices.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 原定计划被室友突发提议（如参加突发社团笔试）打乱时的心态。你是那种会觉得“计划乱了很焦虑”的人，还是会觉得“没准会有惊喜，去看看也行”的随兴派？聊聊你最近一次这种“随风而动”的经历。 `[Signal: Flexibility]`
- **Topic 2:** 准备充分的Pre现场突发故障（如断电、投影坏了）。你是会在台上不知所措，还是能面不改色地开始“脱稿演讲”甚至顺便调弄一下环境气氛？分享一个你化解意外的瞬间。 `[Signal: Composure under chaos]`
- **Topic 3:** 关于生活节奏的偏好。你更喜欢那种“每天按课程表雷打不动”的确定感，还是那种“每天睁眼都期待今天会有什么插曲”的变化感？为什么？ `[Signal: Preference for dynamism]`
- **Topic 4:** 考试时间临时变动导致回家车票作废时的第一反应。是会气死想找人投诉，还是能迅速接受现实，觉得“正好在学校多逛半天，换张票就行”？聊聊你对这种“不可控变数”的容忍度。 `[Signal: Stress response to change]`
- **Topic 5:** 进入一个新环境（如社团或项目组）面对性格迥异的人。你是觉得“环境不合就待不下去”，还是觉得自己能像水一样，慢慢磨合、适应对方的节奏？ `[Signal: Social adaptation]`
- **Topic 6:** 最理想的暑假旅行状态。是那种列满详尽打卡清单并严格执行的，还是只定个出发日子，剩下的全靠“偶遇”和“心情”？聊聊你最舒服的放松方式。 `[Signal: Planning vs Drifting]`

### PG Scenario (研究生场景)
- **Topic 1:** 导师突然告诉你，你做了半年的实验思路被国外大牛刚发的Paper证伪了，需要推倒重来。你第一反应是“毁了毁了”，还是会兴奋地觉得“太棒了，正好我可以试试那个更大胆的新想法”？ `[Signal: Pivot speed]`
- **Topic 2:** 当实验室突然要搬迁，或者整个项目组临时调整了研究方向。你是那个最快调整好状态投入新环境的，还是会怀旧、纠结于“旧路子”很久的人？ `[Signal: Transition ease]`
- **Topic 3:** 面对科研中的变数。如果两个课题：一个是完全确定的、按部就班就能出成果；另一个是充满变数、随时可能需要调转船头的。你会更偏向哪一个？你对这种“模糊性”的容忍度高吗？ `[Signal: Tolerance for ambiguity]`
- **Topic 4:** 关于人际冲突的缓冲。面对那些性格极其多变、甚至有些阴晴不定的导师或合作者，你是否能做到“见招拆招”，而不影响自己的科研心情？ `[Signal: Interpersonal buffering]`
- **Topic 5:** 突发状况下的资源再分配。比如跨组实验时对方设备故障导致你的排班全乱了。你是否能瞬间利用这段空余去处理别的事，而不是陷入焦虑的等待？ `[Signal: Real-time resource management]`
- **Topic 6:** 关于职业规划的笃定感。你是否觉得并不需要太精确的长期规划，因为“只要我当下把每一步走好，变化自然会把我带到该去的地方”？聊聊这种“活在当下”的职场哲学。 `[Signal: Current-moment focus]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 当突发情况打乱我的原有计划时，我不容易感到压力或沮丧。 [ ]
2. 我非常习惯处理各种临时出现的突发需求。 [ ]
3. 我宁愿应对当下的挑战，也不想花太多时间去做长期的僵化计划。 [ ]
4. 随着环境的变化而改变我的行动方式，对我来说是非常自然的。 [ ]
5. 很多人认为我是团队里应对变化最灵活的那个人。 [ ]
6. 我能在混乱复杂的局面中迅速找到平衡感。 [ ]
7. 我认为过度的未雨绸缪反而会错过当下的机遇。 [ ]
8. 我享受那种在变化中即兴发挥、边走边想的过程。 [ ]
9. 我能够轻松地和各种不同背景、不同性格的人打交道。 [ ]
10. “以变应变”是我解决所有问题的核心策略。 [ ]
11. (反向) 我必须严格遵守固定的日程表，否则我会感到极度不安。 [ ]
12. (反向) 哪怕是很小的突发状况，也会让我手忙脚乱很久。 [ ]
13. (反向) 我认为稳定的环境比什么都重要，我讨厌意外。 [ ]
14. (反向) 所有的变数对我来说都是一种风险，而非机会。 [ ]
15. (反向) 我觉得那些没有计划直接动手的人是非常轻率的。 [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：高韧性、高情商、危机处理大师。适合：快消行业、灵活就业、初创项目联络官。
- **负面风险**：可能缺乏长期目标、容易显得“随大流”、对规章制度可能有抵触。
- **岗位匹配暗示**：适合需要不断应对突发状况的岗位，避免需要“十年如一日”枯燥重复的行政岗。
