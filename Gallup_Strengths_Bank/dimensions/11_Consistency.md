# Gallup Theme: Consistency (公平)

## Core Definition
Balance. Trying to treat everyone in the world with the same rules.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 面对有才华但迟到了一小时的面试者。你是会觉得“才气可以抵消规矩”从而给她通融，还是会因为原则问题而坚持“规矩就是规矩，一分钟都不能宽限”？聊聊你对这种“标准化”的执念。 `[Signal: Rule fundamentalism]`
- **Topic 2:** 关于“大锅饭”与“按劳分配”的内心权衡。如果小组作业老师给了所有人一样的分数，你是觉得这样最公平、能减少潜在矛盾而感到舒服，还是非要寻求一套严密的分配规则才行？ `[Signal: Egalitarian preference]`
- **Topic 3:** 制定社团规则时的审美。你是更喜欢那种“充满灵活度、因地制宜”的条文，还是那种像程序执行一样、谁来都绝对一样的“标准操作”？ `[Signal: Standardization drive]`
- **Topic 4:** 对插队者的本能反应。面对排队时的不守规矩者，你是否即便自己不赶时间，也会有强烈的冲动去制止他，只为了维护那点“先来后到”的基本平衡？ `[Signal: Fairness vigilance]`
- **Topic 5:** 对“体制开绿灯”的忍受度。如果你发现辅导员给个别学生特殊优待（如挂科不留底），你会产生一种极其强烈的不适甚至愤慨感吗？你对“公平”的洁癖重吗？ `[Signal: Systemic integrity]`
- **Topic 6:** 关于评判体系的客观性。你更希望你的大学生涯是充满了导师“个性化”的非标准化引导，还是由一套绝对透明、标准化的评分系统来定输赢？ `[Signal: Objectivity vs Individualization]`

### PG Scenario (研究生场景)
- **Topic 1:** 处理实验室里的“劳模”与“混子”。你会不会坚持“必须严丝合缝按分配好的仪器时段来，绝不能因为是劳模就多给他一点点额外便利”？谈谈你对程序正义的死守。 `[Signal: Procedural justice]`
- **Topic 2:** 导师对“宠儿”学生的优待（如论文挂名特权）。你是不是那个即便自己利益没受损，也会觉得这种规则崩坏而带来的行为极其难以接受、甚至想去揭露的人？ `[Signal: Institutional fairness]`
- **Topic 3:** 理想的工作环境规则。你是不是更倾向于在那种规则透明到甚至有点冷冰冰、奖惩分明、绝对不允许“看主观心情办事”的环境下工作？ `[Signal: Environment predictability]`
- **Topic 4:** 学术评审时的“盲审”心态。在评审时，你是否能完全基于客观逻辑抛开对方的“名气”或“人脉背景”，仅仅针对论文本身的数据给分？ `[Signal: Blind evaluation]`
- **Topic 5:** 制定新的KPI考核标准。你会不会花极长的时间去调研每一个岗位的平均工作量，以确保规则对分工不同的每个人都在逻辑上绝对对等？ `[Signal: Detailed equality]`
- **Topic 6:** 天才与普通人的竞争规则。你是否认为，一个好的制度应当强制让“天才”和“普通人”都在同样的规则重力下竞争，而不应有任何基于天赋的额外偏袒？ `[Signal: Structural consistency]`

3. 在处理问题时，我会尽力保持客观和公正，避免受到私人关系的干扰。 [ ]
4. 我发现建立清晰的标准对于团队的高效运作至关重要。 [ ]
5. 重视公平和一致性，是我做人最基本的底色。 [ ]
6. 所有的流程都应当被书面化且严格执行，拒绝“潜规则”。 [ ]

8. 对我来说，给不同的人套用不同的标准是一件非常复杂且错误的事。 [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极高诚信度、优秀的流程设计者、能消除团队内的裙带关系。适合：合规审计、标准化生产管理、质量监督、体制内行政管理。
- **负面风险**：在极度需要灵活应变的复杂市场环境（如BD谈判）中可能显得僵化、可能会压制团队内非对等贡献带来的超额激励需求。
- **岗位匹配暗示**：适合需要“制度化” and “一致性”的稳定赛道。
