# Gallup Theme: Relator (交往)

## Core Definition
Enjoying close relationships with others. You derive great satisfaction from working hard with friends to achieve a goal.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** “小院深处”的社交。比起去参加那种几百人的喧闹派对，你是否宁愿叫上那两个从小玩到大的‘铁瓷’，在深夜的路边摊喝点啤酒、聊聊藏在心里的脆弱？ `[Signal: Depth preference]`
- **Topic 2:** 关系的“托付感”。如果要组建一个小分队去比赛，你是会选那个技术极牛但完全不熟的人，还是会选那个‘默契到甚至一个眼神就能对齐’的老搭档？ `[Signal: Relational trust priority]`
- **Topic 3:** 关系的“自来旧”。你是否一旦认定了一个人，就会迅速把你的‘秘密硬盘’向他敞开，并期待对方也对你毫无保留？你是否迷恋这种灵魂层面的‘投名状’？ `[Signal: Intimacy drive]`
- **Topic 4:** 为哥们“两肋插刀”。你会为了帮一个至交好友处理麻烦，而毅然推掉一个大厂的社交晚宴吗？你是否认为‘深层私人关系’才是生命里最真实、最贵重的东西？ `[Signal: Relational sacrifice]`
- **Topic 5:** 情感的“细腻雷达”。你会留意到好哥们在微信里某个尴尬的停顿、或者异样的语气吗？你是否总在他最需要‘懂他’的那一刻精准出现？ `[Signal: Friendship depth monitoring]`
- **Topic 6:** “生死契阔”的创业梦。你是否认为一个团队最强大的时候，不在于合同有多严密，而在于核心成员之间那种‘背靠背、同生共死’的兄弟情义？ `[Signal: Relational foundation]`

### PG Scenario (研究生场景)
- **Topic 1:** 实验室里的“小堡垒”。大家都在忙着往外跑各种社交，你是否反而更受用于在那一亩三分地里，跟你合作了很久的师兄在那边调代码一边长谈？你觉得这种‘战友情’才是科研能熬下去的唯一动力？ `[Signal: Small-circle loyalty]`
- **Topic 2:** 师徒关系的“情感契约”。导师提出了一个让你很不爽、甚至有点违章的要求，你会因为‘他带了我这么多年，我不能让他失望’这种深层羁绊选择妥协吗？ `[Signal: Personal bond commitment]`
- **Topic 3:** “肤浅社交”的生理不适。参加学术会议时，面对那种客套的‘三分钟谈话’，你会感到巨大的虚无感吗？你是否更怀念那种在实验室私下交流产生的深层连接？ `[Signal: Rejection of superficial networking]`
- **Topic 4:** 信任背后的“疯狂托底”。只要是那个你最信赖的课题组骨干交代的活，哪怕再碎、再脏，你也绝对会短时间内为他做到完美，以此作为对他信任的最高致敬？ `[Signal: Mutual trust-based delivery]`
- **Topic 5:** 关系的“闪电合拢”。当项目组来了一个跟你频率极搭的人，你会不会通过几次深夜长谈，迅速把他发展成‘生死之交’？你是否迷恋这种‘磁场瞬间对齐’的感觉？ `[Signal: Rapid deepening of trust]`
- **Topic 6:** 你的“核心圈子”终极愿想。是否梦想过未来能跟几个最要好的同门一起出去创业、或者在同一个城市‘抱团聚暖’，共同撑起一个共同的学术梦想？ `[Signal: Collective success of the inner circle]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我非常享受与一小群志同道合、且彼此深切了解的人一起工作. [ ]
2. 与结交新朋友相比，我更愿意花时间去加深与老朋友的关系. [ ]
3. 我认为真正的友谊建立在彼此坦诚所有的脆弱和秘密的基础之上. [ ]
4. 在团队中，我通常是那个最被核心成员所信赖、并能托付重任的人. [ ]
5. 看到原本疏离的关系因为我的真诚投入而变得亲密无间，我会感到很有成就感. [ ]
6. 与其参加大型社交聚会，我更喜欢只有两三个人参加的深度长谈. [ ]
7. 我愿意为那些我真正在意的人做出巨大的牺牲，且不求回报. [ ]
8. 对我来说，一个无法建立深厚个人交情的工作环境是极其寒冷且无趣的. [ ]
9. 我能迅速地感知到亲近之人情绪上的每一个微小起伏. [ ]
10. “莫逆之交”和“团队默契”是我追求的核心生命价值. [ ]
11. (反向) 认识更多的各行业大佬比守着几个老朋友对职业发展更有用. [ ]
12. (反向) 我认为在职场中保持适当的距离感才是最专业的做法. [ ]
13. (反向) 过度向人展示自己的秘密是非常不成熟且危险的行为. [ ]
14. (反向) 我觉得所谓的朋友关系在利益变动面前往往是脆弱且不可靠的. [ ]
15. (反向) 我更喜欢在一个大家公事公办、不需要投入太多私人情感的环境里工作. [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极高忠诚度、极强的核心团队粘合力、卓越的长期客户关系维护者、能在信任基础上爆发出惊人的协同生产力。适合：核心合伙人、高端客户经理、研发实验室核心骨干、家族企业。
- **负面风险**：可能会显得有些“拉帮结派”或对外封闭、可能会过度关注关系而失去了对全局利益的判断力、在需要频繁接触陌生人的岗位中会感到精疲力竭。
- **岗位匹配暗示**：适合需要“深度信赖” and “长期协作”的内核岗位。
