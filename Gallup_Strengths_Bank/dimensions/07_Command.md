# Gallup Theme: Command (统帅)

## Core Definition
Taking charge. You have presence and can drive others to a conclusion.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 面对小组讨论陷入三小时无效循环的僵局。你会不会直接在群里打破沉默：“行了！听我的，现在按A方案定，出了问题我抗，散会！”？聊聊你这种强势终结内耗的时刻。 `[Signal: Decisive closure]`
- **Topic 2:** 当你看到“群龙无首”导致的团队低效。你会不会忍不住想跳出来接管全场？那种“看着大家因为犹豫而由于墨迹就生理性烦躁”的冲动，你常有吗？ -> NO! FIX!
- **Topic 2:** 当你看到“群龙无首”导致的团队低效。你会不会忍不住想跳出来接管全场？那种“看着大家因为犹豫不决而浪费时间就生理性烦躁”的冲动，你常有吗？ `[Signal: Leadership vacuum filling]`
- **Topic 3:** 揭穿“国王的新衣”。你会觉得当面指出别人的错误很尴尬吗？还是觉得“既然事实摆在这，说真话哪怕不好听也是对大家负责”？分享一次你硬刚错误的经历。 `[Signal: Direct confrontation]`
- **Topic 4:** 挑战不合理的“大目标”。面对辅导员或权威给出的不合理硬指标，你是那个敢带头去办公室据理力争、甚至要求修改规则的“刺头”吗？聊聊你内心的对抗力量。 `[Signal: Challenging authority]`
- **Topic 5:** 对付团队里的“南郭先生”。当有人试图偷懒或浑水摸鱼时，你是否能用一种不怒自威的气场让他乖乖干活？你享受这种“掌控局势”的威慑力吗？ `[Signal: Enforcement power]`
- **Topic 6:** 关于责任与权力的天平。比起做一个默默无闻的执行者，你是否宁愿掌握指挥权并承担全部风险？那种“一呼百应”的感觉对你意味着什么？ `[Signal: Appetite for power/responsibility]`

### PG Scenario (研究生场景)
- **Topic 1:** 实验室项目落后，师弟师妹还在敷衍了事。你会选择私下婉转提醒，还是会当面严肃指正：“这个标准不行，重来，下次再犯就别进组”？聊聊你对“标准”的强势坚持。 `[Signal: Assertive accountability]`
- **Topic 2:** 当导师在某个关键决策上犹豫不决。你是不是那个敢直接摆出事实、并强势地“反向管理”老板，推动他选择那个你认为正确的方向的人？ `[Signal: Upward influence]`
- **Topic 3:** 面对外部合作方的“不配合”。你是那个敢于拍桌子维护实验室利益、并让对方不得不吐出资源并退步的“狠角色”吗？谈谈你的谈判气场。 `[Signal: Tough negotiation]`
- **Topic 4:** 组会气氛冰冷、数据惨不忍睹时的指挥。你是否敢站出来带头承认失败，并立刻下令切换备选跑道，把大家的节奏强行带回正轨？ `[Signal: Crisis decision-making]`
- **Topic 5:** 关于“沟通成本”的零容忍。你会不会觉得因为性格软弱而不敢当面解决问题的“柔和文化”让你窒息？你更喜欢那种“直球对决”的工作环境吗？ `[Signal: Low tolerance for indirectness]`
- **Topic 6:** 你的领袖气场。在跨学科合作中，你是否有一种天生的自信，能让别人相信“只要跟着你走，这事就一定能成”？这种气场是刻意培养还是天生的？ `[Signal: Natural presence]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我在团队中通常能自然地掌控局面，人们倾向于听从我的意见。 [ ]
2. 我愿意为了达成目标而与他人发生直接的冲突。 [ ]
3. 对我来说，在公开场合表达一个可能会引起争议的观点并不可怕。 [ ]
4. 当情况陷入僵局时，我能果断做出决定并承担相应的后果。 [ ]
5. 我认为领导力的核心在于能够强势地推动项目向前运行。 [ ]
6. 我尊重那些说真话、敢硬碰硬的人，我也努力成为这样的人。 [ ]
7. 我发现自己对“弱势的领导力”非常不能容忍。 [ ]
8. 一旦我确信自己的判断，我就会通过各种手段说服或迫使他人执行。 [ ]
9. 我享受那种在风暴中心指引方向带来的紧迫感和力量感。 [ ]
10. 对我来说，模糊不清的状态比直接的对抗更让我感到不适。 [ ]
11. (反向) 为了维持大家的体面和关系，我会选择隐藏自己的不同意见. [ ]
12. (反向) 当大家争执不下时，我更希望有个更厉害的人来做主，而不是我. [ ]
13. (反向) 我认为让每个人都感到舒服，比快速达成结论更重要. [ ]
14. (反向) 发号施令会让我的内心感到极大的负担和不适. [ ]
15. (反向) 我更喜欢做一个默默完成任务的执行者，而非决策者. [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极高担当、危机决策能力、强大的执行推动力。适合：危机公关、突击项目负责人、高强度商务拓展、纪委/合规官。
- **负面风险**：容易显得咄咄逼人、可能造成团队关系紧张、如果判断失误会带来较大损失。
- **岗位匹配暗示**：适合需要“铁腕”或“前线决策”的位置，避免需要长期细腻情感安抚的纯HR或心理咨询岗。
