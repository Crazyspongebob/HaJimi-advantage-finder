# Gallup Theme: Input (搜集)

## Core Definition
A craving to know more. You like to collect and archive all kinds of information.

---

## [Mode A] Chatty Mode (闲聊模式)

### UG Scenario (本科场景)
- **Topic 1:** 赛博空间的“囤积本能”。你的收藏夹和网盘里，是不是塞满了因为觉得‘以后可能有用’而存下的各种网课、PPT模版或冷门电子书？哪怕你明知道可能这辈子都看不完，但只要它们静静躺在那，你就会感到莫名的安全感？ `[Signal: Archive compulsion]`
- **Topic 2:** 信息捕获的“多巴胺”。在冲浪时发现一个看似无用的冷知识（比如北极熊的毛是透明的），你会不会下意识地想把它记录下来或者分享出去？你是否狂热于这种不断发现并占有新知识碎片的快感？ `[Signal: Curiosity-driven hoarding]`
- **Topic 3:** 物质世界的“标本师”。旅行时，你是那种‘拍两张照打卡’的人，还是非要带回一袋子门票根、宣传页或者是路边那块特别的鹅卵石，并隆重地把它们塞进你那满满当当的手帐本里？ `[Signal: Artifact collection]`
- **Topic 4:** 购物前的“尽职调查”。如果你打算买款新产品，你会不会刷完所有平台的深度评测，非要把其中的成分、品牌历史和所有槽点都钻研透了才肯点下单键？ `[Signal: Exhaustive research]`
- **Topic 5:** 寻找“边缘知识”的诱惑。你会去关注那些极其冷门、甚至只有个位数播放量的专业纪录片或博客吗？你是否觉得越是偏僻、别人看不见的信息，就越具某种莫名的魅力？ `[Signal: Knowledge breadth]`
- **Topic 6:** 你的价值底层。你是否更倾向于认为：一个所谓天才的创新，本质上只是个人资源库里那成千上万个‘备选组件’的一次精准提取与重新组合？ `[Signal: Resource-based problem solving]`

### PG Scenario (研究生场景)
- **Topic 1:** 调研启动的“火力覆盖”。新开一个课题，你是先看两篇核心文献找感觉，还是会废寝忘食地把近二十年所有相关的全文、甚至作者在学术论坛里的一条回复都统统下载到你的EndNote库？ `[Signal: Primary source obsession]`
- **Topic 2:** 实验室的“活百科全书”。你是否被大家公认为那个‘任何搞不定的奇怪Bug、或是不知道去哪买的小众试剂都去问他准没错’的神奇学长/学姐？ `[Signal: Information hub]`
- **Topic 3:** 维护“经验黑匣子”。你会不会把实验室每一次昂贵的实验失败案例、甚至导师在闲聊时透露的一句操作窍门，都专门建个加密文档随时备查？你是一个‘经验碎片’的狂热收藏者吗？ `[Signal: Procedural knowledge storage]`
- **Topic 4:** 对“历史残余”的痴迷。你是否曾因为在实验室角落发现一本上个世纪留下的、落满灰尘的手绘记录本而感到莫名的悸动？你是否能从这些被遗忘的陈旧信息中读出某种未来的演化线索？ `[Signal: Historical residue sensitivity]`
- **Topic 5:** 关于“跨时空提取”的魔法。当你遇到逻辑瓶颈时，你是否能神奇地跳回到你几年前随手划在书边的一行涂鸦，并从中发现破解当下难题的核心钥匙？ `[Signal: Long-term retrieval]`
- **Topic 6:** 你的“万能图书馆”理想。是否梦想过能拥有一间集齐了全世界最晦涩、最新颖、最全面的标本与文献的私人实验室，成为行业知识谱系中最后的一道可靠屏障？ `[Signal: Curatorial ambition]`

## [Mode B] Scale Mode (量表模式 - 1-5)
*1=极其不同意, 2=不太同意, 3=一般, 4=比较同意, 5=极其同意*

1. 我天生就对获取新的信息、知识或实物充满了渴望。 [ ]
2. 我非常享受整理、归类和存储各种资源的过程。 [ ]
3. 我认为掌握的信息越多，解决问题的能力就越强。 [ ]
4. 这种搜集的过程能带给我极大的安全感和充实感。 [ ]
5. 很多人认为我是一个博学、广泛查阅资料且非常有储备的人。 [ ]
6. 与其直接寻找答案，我更喜欢先在自己的庞大库房里搜索一番。 [ ]
7. 我对各种琐碎的信息都保有一种近乎孩子般的好奇心。 [ ]
8. 对我来说，一个无法被追溯来源的数据是极度不可信的。 [ ]
9. 我能迅速地从海量信息中捞出我需要的那一块拼图。 [ ]
10. “博闻强记”和“有备无患”是我应对所有不确定性的核心策略。 [ ]
11. (反向) 搜集太多无用的东西只会浪费我的硬盘空间 and 大脑内存。 [ ]
12. (反向) 我认为掌握核心那10%的信息就够了，其他的都是噪音。 [ ]
13. (反向) 我讨厌整理资料，觉得那是非常枯燥且低级的工作。 [ ]
14. (反向) 碰到问题我会直接去问专家，而不是自己闷头查资料。 [ ]
15. (反向) 我觉得那些整天收藏这收藏那的人，其实是因为他们自己缺乏原创思考。 [ ]

## [Module C] Signal Interpretation (信号解读)
- **正面特征**：极佳知识储备、优秀的资源挖掘者、能为团队提供源源不断的素材支撑。适合：资深研究员、创意文案、大数据挖掘初步清洗、博物馆策展人、新闻编辑。
- **负面风险**：可能会陷入“数字囤积症”而忽略了应用、可能会信息过载而在决策时感到纠结、可能会让行动变缓。
- **岗位匹配暗示**：适合需要“先搜集后加工”的慢节奏、高储备行业。
