# Gallup CliftonStrengths Bank v2 (Redesign)

## Project Overview
This repository contains a comprehensive, redesigned question bank and matching logic for Gallup CliftonStrengths, specifically tailored for Chinese undergraduate and graduate students.

## Repository Structure
- `dimensions/`: 34 MD files for each Gallup strength.
- `industries/`: China-specific industry maps and interest questions.
- `logic/`: UG/PG branching logic and AI analysis scoring systems.
- `matching/`: Career-to-strength matching principles.
- `logs/`: Sub-agent design logs and research history.
- `README.md`: This file.
- `Super_Bank_v2.md`: Consolidated master file.

## Sub-agent Framework (Phase 2 Extension)
1. **Scale Designer**: 1-5 Likert scale development.
2. **Scenario Expander**: Multi-scenario situational drafting.
3. **Tone Polisher**: Maintaining chatty/informal UX.
4. **Data Architect**: Structured file formatting.
5. **Cross-Domain Verifier**: Ensuring 4-domain balance.
6. **Career Mapping Expert**: Aligning new questions to job roles.
7. **Industry Expander**: Broadening target sector questions.
8. **Master Compiler**: v3 Consolidation.
